# Graph-Based Injectr Architecture

## Overview

**Graph-Based Injectr** is an AI-powered prompt injection and jailbreaking testing framework built with **LangGraph** for declarative graph-based agent orchestration. It uses a multi-agent architecture with specialized agents for reconnaissance, injection, analysis, and strategic management.

## LangGraph Benefits

1. **Declarative Workflow**: Define agent logic as a graph, not imperative code
2. **Built-in State Management**: Automatic state accumulation across nodes
3. **Streaming**: Native `astream()` for real-time event handling
4. **Conditional Routing**: Flexible branching based on state
5. **Checkpointing**: Optional persistence for long-running campaigns
6. **Debugging**: Graph visualization and LangSmith tracing integration
